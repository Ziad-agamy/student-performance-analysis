Students Performance Dataset Metadata

This document presents the metadata for the `Students_Performance.csv` dataset, categorizing features as either numeric or categorical.

Numeric Columns

Numeric columns contain quantitative, measurable data, which can be continuous or discrete.

  • Age:

    The student's age.

  • Grade:

    The student's academic grade level.

  • TestScore_Math:

    The student's test score in mathematics.

  • TestScore_Reading:

    The student's test score in reading.

  • TestScore_Science:

    The student's test score in science.

  • GPA:

    The student's Grade Point Average.

  • AttendanceRate:

    The student's attendance rate.

  • StudyHours:

    The number of hours a student studies.

  • FreeTime:

    The amount of free time a student has.

  • GoOut:

    The frequency with which a student goes out.

Category Columns

Categorical columns represent qualitative data, denoting discrete groups or classifications.

  • Gender:

    The student's gender (e.g., Male, Female).

  • Race:

    The student's race or ethnicity.

  • SES_Quartile:

    The student's socioeconomic status, represented by a quartile.

  • ParentalEducation:

    The highest level of education attained by the student's parents.

  • SchoolType:

    The type of school the student attends (e.g., Public, Private).

  • Locale:

    The school's location (e.g., Rural, Suburban, City, Town).

  • InternetAccess:

    A binary indicator of whether the student has internet access (1 = Yes, 0 = No).

  • Extracurricular:

    A binary indicator of whether the student participates in extracurricular activities (1 = Yes, 0 = No).

  • PartTimeJob:

    A binary indicator of whether the student has a part-time job (1 = Yes, 0 = No).

  • ParentSupport:

    A binary indicator of whether the student has parental support (1 = Yes, 0 = No).

  • Romantic:

    A binary indicator of whether the student is in a romantic relationship (1 = Yes, 0 = No).
